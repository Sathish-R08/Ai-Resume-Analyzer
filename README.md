# RezuMatch.Ai - Resume Analyzer 

A production-ready, full-stack application designed to help job seekers and recruiters automatically assess resumes against job criteria. By leveraging Google's Gemini AI framework, the app instantly parses PDF resumes to provide in-depth feedback, ATS (Applicant Tracking System) scoring, and targeted improvement tips.

### 🚀 Tech Stack Highlights
* **Frontend:** React, React Router v7, Tailwind CSS, Zustand (State Management)
* **Backend:** Node.js, Express.js
* **Database & Auth:** MongoDB Atlas, secure JWT Authentication
* **Artificial Intelligence:** Google Gemini API (gemini-2.5-flash)
* **Deployment:** Vercel (Frontend) & Render (Backend)

### ✨ Core Features
* **Authentication:** Secure user sign-up and login system.
* **PDF Parsing:** Securely reads and extracts text from uploaded PDF resumes.
* **AI Analysis:** Analyzes the resume against job requirements to calculate a match score and provide actionable improvement feedback.
