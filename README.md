# RezuMatch.Ai

Build an AI-powered Resume Analyzer using React, Node.js, and MongoDB. This project enables seamless user authentication via JWT, resume uploads, and smart AI-based resume-to-job matching with personalized feedback and ATS (Applicant Tracking System) scores powered by the Gemini AI API—all wrapped in a clean, reusable UI.

---

## 1. Features

- Express & MongoDB custom authentication system with JWT sessions.
- Upload and analyze resumes securely inside the browser via IndexedDB.
- Match resumes to job descriptions using Google Gemini AI and receive tailored ATS scores with structured feedback.
- Clean, responsive, and modular user interface built for maintainability and consistency.
- Reusable components and a scalable architecture for efficient full-stack development.
- Fully responsive design that works across desktops, tablets, and mobile devices.
- Built using modern UI/UX design practices with Tailwind CSS.

---

## 2. Tech Stack

### Frontend Frameworks and Libraries

- **React**: A popular JavaScript library for building dynamic user interfaces using components and a virtual DOM.
- **React Router v7**: Advanced routing library offering nested routes and SPA functionality.

### Backend and AI Tools

- **Node.js & Express**: A fast, minimal backend to manage secure REST routes and authentication.
- **MongoDB Atlas**: A managed NoSQL database acting as the primary store for user credentials.
- **Google Gemini API**: Advanced LLM processing ensuring fast, intelligent resume analysis and scoring.

### UI, State, and Styling

- **Tailwind CSS**: A utility-first CSS framework that allows rapid UI development with low-level utility classes.
- **Zustand**: A minimal state management solution for React that provides global state control for authentication and caching without boilerplate.
- **Vite**: A high-performance build tool and development server with native ES module support and fast HMR.
